# 音效文件说明

## 文件夹结构
```
sounds/
├── reminders/    # 太乙真人提醒音效
└── README.md     # 本说明文件
```

## 音效文件要求

### 提醒音效 (reminders/)
- 文件格式：MP3
- 建议时长：2-5秒
- 建议大小：<500KB
- 命名规则：reminder-1.mp3, reminder-2.mp3, ...
- 音量：标准化到 -6dB

## 使用说明
1. 将音效文件放入对应文件夹
2. 文件名必须使用英文和数字
3. 建议对音频进行如下处理：
   - 剪辑掉开头和结尾的静音部分
   - 标准化音量
   - 压缩文件大小
